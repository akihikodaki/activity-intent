Abstract

This document lists specifications to implement intent feature for ActivityPub.

Status of This Document

This section describes the status of this document at the time of its publication. Other documents may supersede this document.

While specifications listed in this document will eventually be proposed for W3C standard, this document itself is not included in those proposals.

Listed specifications are published by the author as a proposal for Mastodon and other [ActivityPub] applications. Please write comments regarding this document to the following issue in Mastodon:
URL scheme for remote follow, share buttons · Issue #2291 · tootsuite/mastodon

This document is expected to be useful also for other ActivityPub applications. If you have any request to change for them, please contact the author. E-mail is the most reliable means for that, but OStatus2 is also acceptable. Send a note to @fn_aki@pawoo.net.

Specifications

The following specifications are proposed to implement intent feature for ActivityPub.

  • ActivityPub Extension for Intent
  • URL Scheme for Activity Streams 2.0
  • Intent Class in Activity Streams 2.0 Vocabulary
